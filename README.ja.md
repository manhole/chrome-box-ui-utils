# Box UI Utils

[English](README.md)

[Box](https://www.box.com/) のフォルダパスを省略なしで表示し、クリップボードにコピーできる Chrome 拡張機能です。

## 動機

Box のパンくずリストは、フォルダ階層が深くなると中間フォルダをオーバーフローメニューの中に隠してしまいます。フルパスがひと目で分からず、他の人に共有するためにコピーすることもできません。

## 解決策

Box UI Utils は、オーバーフローメニューに隠れたフォルダも含めてパンくずリストを完全に取得し、画面下部のフローティングバーに表示します。ワンクリックでパスをクリップボードにコピーできます。

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions` を開く
3. 右上のデベロッパーモードを有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、リポジトリのディレクトリを選択

## 使い方

Box のフォルダページ (`https://app.box.com/folder/...`) を開くと、画面下部にフルパスが表示されます。

```
ProjectA > Design > 2024 > Q4 > Final Review
```

クリップボードアイコンをクリックするとパスがコピーされます。

## 仕組み

- `https://*.box.com/*` のページにコンテンツスクリプトを注入
- パンくず要素 (`.ItemListBreadcrumb`) を `MutationObserver` で監視
- オーバーフローメニューをプログラムで展開し、非表示の親フォルダを取得
- ルートフォルダ (「ファイル」等) は `/folder/0` の判定で除外
- 現在のフォルダ名はページ見出しからフォールバック取得

## 制限事項

- Box の現在の DOM 構造 (`ItemListBreadcrumb` 等のクラス名) に依存しています。Box の UI 更新により動作しなくなる可能性があります。
- 初回読み込み時、非表示フォルダの取得のためにオーバーフローメニューが一瞬開閉します。
- フォルダビュー (`/folder/{id}` URL) のみ対応しています。
